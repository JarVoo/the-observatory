# Data Sources

## Gibraltar
### Gibraltar Statistics Office (GSO)
- Website: https://www.gibraltar.gov.gi/statistics
- Covers: visitor numbers, cruise arrivals, frontier crossings
- Format: PDF reports and Excel tables
- Frequency: Annual and quarterly

## Campo de Gibraltar
### Instituto Nacional de Estadística (INE)
- Website: https://www.ine.es
- Covers: population, employment, economic indicators
- Format: API and downloadable tables
- Frequency: Annual

### Junta de Andalucía
- Website: https://www.juntadeandalucia.es
- Covers: regional tourism statistics
- Format: PDF reports
- Frequency: Annual

## Notes
- GSO data requires manual download — no public API
- INE has a free API — automation possible
- Campo municipal level data is sparse
