# The Observatory

A data initiative monitoring tourism, visitor impact, and economic 
trends across Gibraltar and the Campo de Gibraltar region.

## Focus Areas
- Tourism & visitor numbers
- Cross-border movement
- Economic impact of tourism

## Geographic Scope
- Gibraltar
- Campo de Gibraltar (Algeciras, La Línea, San Roque, 
  Los Barrios, Tarifa, Castellar de la Frontera, 
  Jimena de la Frontera)

## Repository Structure
- data/       raw and processed datasets
- scripts/    Python scripts for data collection
- reports/    analysis outputs and summaries
- docs/       methodology and data source notes

## Data Sources
- Gibraltar Statistics Office (GSO)
- Instituto Nacional de Estadística (INE)
- Junta de Andalucía
- Local council publications

## Status
In development — tourism data collection in progress.
